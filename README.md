# MovieBot

MovieBot — это личный телеграм-бот для фильмов и сериалов.  
Он помогает быстро вести список просмотренного, искать что посмотреть дальше и делать рекомендации на основе ваших вкусов.

## Что умеет бот

- Добавлять фильмы и сериалы в ваш список просмотренного.
- Понимать голосовые сообщения и превращать их в аккуратную запись.
- Если данных не хватает, бот сам попросит уточнить текстом или голосом.
- Распознавать постеры по фото и предлагать быстрое добавление.
- Давать персональные рекомендации по вашим оценкам и жанрам.
- Показывать случайный фильм, который вы еще не смотрели.
- Искать по названию, жанру, владельцу подборки.
- Показывать последние добавления, топ по оценкам и статистику.
- Считать «победителя месяца» (муж/жена) по качеству и активности.
- Показывать расход AI-токенов отдельно по Gemini и Mistral.

## Как обычно пользоваться

1. Открыть бота в Telegram.
2. Пользоваться нижними быстрыми кнопками:
   `Рекомендовать`, `Добавить`, `Последние`, `Случайный`, `Статистика`, `Меню`.
3. Для добавления фильма:
   - либо пройти пошаговый диалог;
   - либо надиктовать голосом;
   - либо отправить постер.
4. Перед сохранением бот показывает карточку для проверки.
5. Нажать «Сохранить» или «Отмена».

## Голосовое добавление

Можно говорить свободно, как удобно.  
Если бот не понял часть информации, он не прерывает процесс, а предлагает:

- уточнить текстом;
- отправить еще одно голосовое;
- отменить.

## Рекомендации

Рекомендации строятся по вашему списку и оценкам, а не случайно.  
Бот старается предлагать то, что реально близко по жанру и настроению, и не повторять уже просмотренное.

## Для чего это удобно

- Не теряются впечатления о фильмах.
- Легко договориться «что смотреть сегодня».
- Видно, как меняются вкусы со временем.
- Вся история в одном месте, прямо в Telegram.
